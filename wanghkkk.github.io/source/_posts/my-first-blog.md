## hexo安装

### 安装依赖：nodejs

#### 通过nvm来安装管理nodejs

安装nvm：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | sh
```

通过nvm安装node：

```bash
num install node
```

安装hexo工具：
```bash
npm install -g hexo-cli
```
